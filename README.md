# VGA36x24 v0.1

Copyright 1994 Maxim Masiutin. All rights reserved.

A graphics library for the VGA 360x240 mode (also known as ModeX).

This version of the library is written by Max Masyutin (Maxim Masiutin) on November 2, 1994, using assembly language for Intel 80186 processors (Turbo Assembler), and has an interface unit for Turbo Pascal.

There is another, lighter library, called XML36x24 which is based on VGA36x24. See <https://github.com/maximmasiutin/xml36x24> for the XML36x24 repository.
